🐧 Operating System Shell Scripting Project
📌 Project Overview
This project consists of a collection of Bash shell scripts designed to automate common operating system tasks such as file handling, system monitoring, user interaction, and process management. It demonstrates practical implementation of Linux commands and scripting concepts.

▶️ How to Run the Scripts
Step 1: Open Terminal (Ubuntu / CMD with WSL)
Step 2: Navigate to Project Folder
cd path/to/project
Step 3: Give Permission
chmod +x script1.sh
Step 4: Run Script
./script1.sh

⚠️ Common Errors & Fixes
❌ Permission Denied
chmod +x filename.sh
❌ No such file or directory
•	Check filename
•	Use ls to verify file exists
❌ Command not found
•	Ensure correct syntax
•	Install required packages
